# Fórmula de Bhaskara em Java com OOP

O projeto apresenta a implementação da fórmula de Bhaskara de forma computacional,
utilizando a linguagem Java.

O projeto foi desenvolvido com finalidade educacional, com o objetivo de praticar
e aprofundar os conceitos de orientação a objetos (OOP) utilizando a linguagem Java.

## Estrutura do projeto

- `Principal`
  Contém a classe responsável pela execução do código.

- `source/`
  Contém a classe responsável pelos cálculos da fórmula de Bhaskara.
